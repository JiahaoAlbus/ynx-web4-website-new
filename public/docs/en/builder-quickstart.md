# Builder Quickstart

YNX Web4 supports three builder paths.

## EVM developers

Use the EVM RPC:
```text
https://evm.ynxweb4.com
```

Chain ID:
```text
9102 / 0x238e
```

## AI agent developers

Use the AI Gateway:
```text
https://ai.ynxweb4.com
```

Core surfaces:
- `/ai/jobs`
- `/ai/vaults`
- `/ai/payments`
- `/x402/resource`

## Web4 developers

Use the Web4 Hub:
```text
https://web4.ynxweb4.com
```

Core surfaces:
- `/web4/policies`
- `/web4/policies/:id/sessions`
- `/web4/agents`
- `/web4/intents`
- `/web4/audit`
