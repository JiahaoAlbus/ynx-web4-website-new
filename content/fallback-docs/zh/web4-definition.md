# WEB4 在 YNX v2 中的定义

状态：active  
最后更新：2026-02-25

## 1. YNX 对 Web4 的定义

YNX v2 的 Web4 不是“给 Web3 套一个 AI 前端”，而是：

- AI Agent 成为一等网络参与者
- 用户以“意图”驱动交互，而非只靠手动点交易
- AI 任务执行、结果、争议、结算在链上有可验证闭环
- 治理与经济参数持续去中心化可审计

## 2. YNX 的 Web4 六个原语

- 身份原语（Identity）
- Agent 原语（Agent Registry）
- 意图原语（Intent）
- 结果声明原语（Claim）
- 挑战原语（Challenge）
- 终局结算原语（Finalize/Slash）

## 3. 对外接口

- AI 结算网关：`/ai/*`
- Web4 协调中心：`/web4/*`
- 链定位和治理元数据：`/ynx/overview`
